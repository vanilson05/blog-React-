# 📘 Blog em React

Projeto de blog desenvolvido com **React**, com foco em aprendizado, organização de componentes e boas práticas no front-end.

Este projeto foi iniciado utilizando o **Create React App** e está em constante evolução conforme novos conhecimentos são aplicados.

---

## 🚀 Tecnologias utilizadas

- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Node.js
- Yarn / NPM

---

## 📂 Estrutura do projeto

```bash
blog/
├── public/
├── src/
│   ├── components/
│   ├── pages/
│   ├── App.js
│   └── index.js
├── package.json
└── README.md


## 🧰 Pré-requisitos

Antes de rodar o projeto, você precisa ter instalado em sua máquina:

- **Node.js** (versão LTS recomendada)  
  👉 https://nodejs.org/
- **NPM** (instalado junto com o Node)
